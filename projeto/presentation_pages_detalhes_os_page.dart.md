Pagina de visualizacao detalhada de uma OS.

Exibe todos os dados de uma Ordem de Servico.

- flutter/material.dart
- checkos/data/models/os_model.dart

# Secoes
- Dados principais (cliente, servico)
- KM (inicial/final)
- Horarios
- Funcionarios responsaveis
- Pendencias
- Relatorio tecnico
- Assinatura
- Imagens
- Botoes de acao: editar, excluir, adicionar diario

# Integracao
- OsModel: dados da OS
- DiarioRepository: diarios relacionados
- Firebase Storage: imagens
