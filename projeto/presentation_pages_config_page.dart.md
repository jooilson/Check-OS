# Descricao Geral
Pagina de configuracoes do aplicativo.

# Responsabilidade no Sistema
Permite ao usuario configurar opcoes do app.

# Funcionalidades
- Alterar tema (claro/escuro)
- Editar dados da empresa
- Gerenciar notificacoes
- Exportar dados
- Ver informacoes do app

# Integração
- ThemeProvider: alterar tema
- CompanyRepository: dados da empresa
- ImportExportService: exportar dados
