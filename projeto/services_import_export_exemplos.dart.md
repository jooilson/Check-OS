# Descricao Geral
Arquivo de exemplos para utilizacao do servico de importacao e exportacao.

# Classe Principal
- **ImportExportExemplos**: Classe com metodos estaticos de exemplo

# Metodos de Exemplo
- **exemplo1_exportarJSON()** - Exporta dados em formato JSON
- **exemplo2_exportarCSV()** - Exporta dados em formato CSV
- **exemplo3_importarDados()** - Importa dados de arquivo
- **exemplo4_listarBackups()** - Lista backups disponiveis
- **exemplo5_deletarBackup()** - Deleta um backup
- **exemplo6_exportarPorPeriodo()** - Exporta dados por periodo
- **exemplo7_obterCaminhoDocumentos()** - Obtem caminho de documentos
- **exemplo8_limparDadosLocais()** - Limpa dados locais
- **exemplo9_sincronizarComServidor()** - Sincroniza com servidor
- **exemplo10_backupAutomatico()** - Fluxo de backup automatico

# Extensoes
- **OsModelExtension**: Extensao para conversao de OS para CSV
